Pictures!! Using the img tag and background-image css property

Websites are more than just text. We can include images in our html documents in two ways:

  1. By using the img tag to include images in the flow of our document
  2. By using the background-img css property to include an image as the background of a block element

Typically, images used as page content are included as image tags, and images used to decorate the background are included using the background image property. It is important to include any images that inform the reader about page content using the img tag, since this allows us to set an alt attribute for screen-readers.
